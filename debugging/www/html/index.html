앱 소개

웹뷰 디버깅/테스트 앱은 모바일 웹 프론트 개발자와 퍼블리셔를 위한 필수 도구입니다.

Android 및 iOS의 웹뷰 환경에서
실시간으로 웹 화면 UI 확인하며,
다양한 디버깅 및 테스트를 진행할 수 있습니다.

주요 기능

  • 상단 상태바 영역과 하단 네비게이션바 영역 확장 기능
  • 웹 페이지 소스 보기 기능
  • 쿠키/로컬/세션 스토리지 조회 및 삭제 기능
  • 자바스크립트 실행 기능
  • 콘솔 로그 보기 기능
  • 네이티브 연동 기능 (자체 하이브리드 솔루션 적용)

사용 사례

  • 다양한 디바이스에서 반응형 웹 개발 및 디버깅
  • 전체 화면에서 웹 화면 UI 테스트

광고 제거

  • 앱에 표시되는 광고를 클릭하여 5초 이상 보시면, 하루 동안 광고가 제거됩니다.

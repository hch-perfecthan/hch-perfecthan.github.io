웹뷰 디버깅 방법

PC의 크롬과 사파리 브라우저에서
안드로이드 및 iOS 웹뷰 디버깅 방법

안드로이드 웹뷰 디버깅 (PC 크롬 브라우저)

  1. 안드로이드 디바이스 USB 디버깅 설정
    - 설정 > 휴대전화 정보 > 소프트웨어 정보에서 빌드번호 연속 터치 후 개발자 모드 활성화
    - 설정 > 개발자 옵션 > USB 디버깅 활성화
  2. PC와 안드로이드 디바이스 연결
  3. PC 크롬에서 chrome://inspect/#devices 실행
  4. 웹뷰 디버깅/테스트 앱 실행 및 웹 페이지 접속
  5. Devices > Remote Target에서 해당 웹 페이지의 Inspect 클릭

iOS 웹뷰 디버깅 (PC 사파리 브라우저)

  1. iOS 디바이스에서 웹 인스펙터 설정
    - 설정 > 사파리 > 고급 > 웹 인스펙터 설정 활성화
  2. 맥 PC에서 사파리 브라우저 실행
  3. 상단 메뉴바에서 사파리 메뉴 > 설정 > 고급 > 메뉴 막대에서 개발자용 메뉴 보기 체크
  4. 맥 PC와 아이폰/아이패드 디바이스 연결
  5. 웹뷰 디버깅/테스트 앱 실행 및 웹 페이지 접속
  6. 상단 메뉴바에서 개발자용 메뉴 > 디바이스명 > 해당 웹 페이지 URL 선택
